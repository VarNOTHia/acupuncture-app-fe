# Acupuncture App Frontend

## 描述
一个全栈的治疗软件。

## 开始

### Installation
```bash
# Clone the repository
git clone [repository-url]

# Install dependencies
pnpm install
```

### 🔧 环境变量配置

请复制项目根目录下的 `.env.example` 为 `.env` 文件，并填写你自己的配置：

```bash
cp .env.example .env
```

本项目使用了高德地图 API，若有需要请自行申请。

### Development
```bash
# Start development server
pnpm dev
```
